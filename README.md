# ryanprescott.com

Personal site built with [Astro](https://astro.build).

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
```

Output goes to `dist/`. Works on any static host (Vercel, Netlify, Cloudflare Pages, etc).

## Stack

- Astro
- Vanilla CSS with custom properties for theming
- Press Start 2P + Cormorant Garamond + Lora fonts
